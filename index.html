tinymce4-mathslate

This is the Mathslate editor adapted as a standard plugin for TinyMCE4. In order to install Mathslate into a local installation of TinyMCE4, copy plugins/mathslate to tinymce/js/tinymce/plugins/mathslate. Otherwise you may run TinyMCE4 from CDN by copying this directory to a web server.

Mathslate is licensed is copyright 2014 by Daniel Thies dthies@ccal.edu and licensed under GPL version 3. Mathslate was originally developed for use in the Moodle platform. These files are a crude adaptation from the Moodle TinyMCE plugin.
